# FaceEmotino
a app can identify people face emotion,flask,python,html,css
## 以下是构建运行步骤
```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python app.py
http://127.0.0.1:5000
```
浏览器输入 http://127.0.0.1:5000
## 如果你想打包成exe
```bash
pyinstaller YourAppName.spec
```

输入完这行指令后
exe文件在dist文件夹里面
